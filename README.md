# Labyrinth - Multiplayer Demo

React + Vite frontend, Spring Boot backend, Docker Compose.
Public demo for portfolio and collaborative development.

## Running locally
# Backend
cd backend
./mvnw spring-boot:run

# Frontend
cd frontend
npm install
npm run dev

## Using Docker Compose
docker compose up --build

## Contributing
Please use feature branches and open PRs. Protected branch `master` requires at least one approval.
